# 生词本 PWA 使用说明

> 一个为你量身定做的英语生词学习小 APP — 录入、复习、开车听学、自动掌握管理。

## 一、如何运行(本地预览)

由于浏览器对 Service Worker 有"必须 HTTPS 或 localhost"的安全限制,**不能直接双击 `index.html` 打开**(那样 PWA 安装功能会失效)。请用下面任一种方式启动一个本地服务器:

### 方法 1:用 Python(电脑上几乎都自带)
打开终端/命令行,进入 `生词本` 文件夹后执行:
```
python3 -m http.server 8080
```
然后浏览器打开:`http://localhost:8080`

### 方法 2:用 Node.js
```
npx serve .
```

### 方法 3:部署到免费空间(推荐,这样手机也能用)
把整个文件夹上传到下面任一平台,会得到一个 `https://xxx.xxx` 的网址,手机浏览器打开它就能用,且能装到桌面:
- **Vercel**:vercel.com,拖文件夹上传即可
- **Netlify**:netlify.com,拖文件夹上传即可
- **GitHub Pages**:把文件夹 push 到 GitHub 仓库,在仓库 Settings → Pages 启用

## 二、添加到手机桌面(成为真正的 App)

### iPhone (Safari)
1. Safari 打开网址
2. 点底部"分享"按钮
3. 选择"添加到主屏幕"
4. 桌面会出现"生词本"图标,和真 App 一模一样

### Android (Chrome)
1. Chrome 打开网址
2. 右上角菜单 → "安装应用" 或 "添加到主屏幕"
3. 桌面出现独立图标

## 三、功能一览

| 功能 | 说明 |
|---|---|
| 录入新生词 | 输入英文 → 自动查询音标、英文释义、例句 → 自动翻译成中文 → 保存 |
| 卡片式复习 | 翻卡测试,4 档评价(还不熟 / 有点印象 / 记住了 / 完全掌握) |
| 艾宾浩斯曲线 | 默认 1/2/4/7/15/30 天间隔自动安排复习 |
| 自动掌握 | 连续答对 5 次(可调)自动移出复习库,只在"已掌握"分类查看 |
| 手动掌握/恢复 | 词库页 ✓ 按钮立即掌握;已掌握的可一键恢复学习 |
| 🚗 开车听学 | 自动连续播放,免手免眼,语音控制,15 分钟自动暂停 |
| 语速调节 | 0.5x ~ 2.0x,默认 0.8x |
| 语音控制 | 说"重复 / 下一个 / 暂停 / 继续"即可控制(开车模式内开启) |
| 每日提醒 | 浏览器通知,设定时间提醒今天有几个待复习 |
| 数据导入导出 | JSON 格式备份,可在新设备恢复 |
| 完全离线 | 第二次打开后断网也能用,数据存在你设备上 |

## 四、开车场景使用建议

1. **出发前**:把手机放在车载支架上,打开生词本 → 点"开车听学模式"
2. **启动**:点中央 ▶ 按钮 → 听到"5 秒后开始" → 把手机放好
3. **驾驶中**:全程不用看屏幕,系统会循环播报"单词→释义→例句"
4. **想重听**:说"重复"(需先开启语音控制)或按耳机/方向盘的"上一首"键
5. **15 分钟后**:自动暂停,防止下车忘关
6. **真正掌握的复习**:留到下车后,在"卡片复习"里手动评价

## 五、数据安全

- 所有单词和设置只保存在你自己设备的浏览器里(localStorage)
- 不上传任何服务器,不需要注册账号
- 清浏览器缓存会丢数据,**建议定期在设置里"导出 JSON"备份**
- 换设备/换浏览器:导出 → 在新设备导入

## 七、常见问题

**Q: 词典查询失败?**
A: 用的是免费 dictionaryapi.dev,偶尔慢或被限速,可以手动填释义后保存。

**Q: 中文翻译不准?**
A: 用的是免费 Google 翻译公共接口,可能不太准。释义框可以手动改。

**Q: 没有声音?**
A: 第一次使用浏览器 TTS 需要用户先点击页面才能发声(浏览器策略),点击页面任意地方再播放即可。

**Q: 安卓 Chrome 不显示安装提示?**
A: 必须通过 https 访问(部署到 Vercel/Netlify),且第一次访问时浏览器会判断是否符合 PWA 标准。如果没出现"安装"按钮,可手动从右上角菜单选"添加到主屏幕"。

**Q: 开车模式语速太快/太慢?**
A: 设置里调"朗读语速",默认 0.8x,可改成 0.6x 或 1.0x。
